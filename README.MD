# Sat2v - Калькулятор производственных цепочек для Satisfactory

## Описание проекта

**Sat2v** — это приложение-калькулятор для игры Satisfactory, которое помогает рассчитывать производственные цепочки и необходимые ресурсы для создания различных предметов.

## Основной функционал

### Главное меню
- **Старт** — запуск расчета производственной цепочки
- **Рецепты** — управление рецептами (в разработке)
- **Настройки** — настройки интерфейса (в разработке)

### Расчет производственных цепочек
Приложение позволяет:
- Выбрать рецепт из списка доступных (ротор, винт, железный прут, железный слиток)
- Указать желаемый выход продукции в минуту
- Рассчитать необходимые ресурсы и производственные цепочки

## Структура проекта

- **Main.py** — главный файл с GUI интерфейсом (tkinter)
- **Calculate.py** — модуль для расчета производственных цепочек
- **Draw.py** — модуль для визуализации (в разработке)
- **recipes.json** — база данных рецептов с информацией о:
  - Ингредиентах
  - Выходе продукции
  - Типе здания (Assembler, Constructor, Smelter)

## Технологии

- Python 3
- tkinter — для графического интерфейса
- matplotlib — для визуализации (запланировано)
- JSON — для хранения данных рецептов

## Текущие рецепты

- **Ротор** — требует железный прут и винт
- **Винт** — производится из железного прута
- **Железный прут** — производится из железного слитка
- **Железный слиток** — производится из железной руды

## Статус разработки

Проект находится в активной разработке. Реализован базовый интерфейс и логика расчета производственных цепочек.
